# Chest X-ray Classification:

## Overview
Developed a deep learning system to classify chest X-rays into COVID-19, Non-COVID Pneumonia, and Normal.

## My Contribution
- Trained EfficientNet-B0 model
- Implemented ensemble learning
- Used Grad-CAM for interpretability
- Worked on U-Net++ segmentation

## Tech Stack
Python, PyTorch, CNNs

## Results
- Accuracy: ~94%
- IoU (segmentation): ~0.73

## Note
Code is not public due to academic collaboration, but approach and results are documented here.
